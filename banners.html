Cómo exportar los banners
1. Abre esta página en Chrome a pantalla completa (1920px de ancho mínimo).
2. Haz clic en "Copiar CSS" para obtener los estilos de cada banner.
3. Para capturar como PNG: abre DevTools → selecciona el .banner-frame → clic derecho → Capture node screenshot.
4. Alternativa: usa una extensión como GoFullPage o la herramienta de recortes del sistema.
5. Sube la imagen a Shopify en Online Store → Themes → Customize → Image banner.
